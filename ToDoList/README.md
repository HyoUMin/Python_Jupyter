# Python CLI To-Do List 📝

간단한 명령줄 인터페이스(CLI) 기반의 할 일 관리 프로그램입니다.
파이썬의 핵심 자료구조인 리스트와 딕셔너리를 활용하여 할 일 데이터를 관리하며, 사용자의 입력 오류를 방지하기 위한 예외 처리 로직이 포함되어 있습니다.

## 💡 주요 기능 (Features)

- 전체 할 일 목록 조회: 저장된 모든 할 일과 날짜를 한눈에 확인합니다.
- 날짜별 할 일 조회: 특정 날짜(예: 2026-04-02)를 검색하여 해당 일자의 할 일만 모아봅니다.
- 새로운 할 일 추가: YYYY-MM-DD 형식의 날짜 검증을 거쳐 새로운 일정을 등록합니다.
- 기존 할 일 수정: 검색된 목록의 인덱스 번호를 선택하여 내용을 업데이트합니다.
- 등록된 할 일 삭제: 완료되었거나 불필요한 일정을 리스트에서 안전하게 제거합니다.

## 🛠 사용 기술 및 환경 (Environment)

- Language: Python 3.10 이상 (match-case 제어문 사용)
- Architecture: 리스트 내부에 딕셔너리를 담는 구조를 활용하여 데이터의 속성(날짜, 내용)을 명확하게 분리
